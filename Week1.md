# 1️⃣ 개념 정리 
## 01. 데이터를 분석하다

```
✅ 학습 목표 :
* 데이터에 대해 이해한다.
* 데이터 분석에 대해 이해한다.
```
### **데이터**  
: **수치의 모음** (단, 문자로 이루어진 데이터도 존재)  
*ex) 고정밀 계측 데이터(생물,의학분야) / 사용자의 검색이력(비즈니스분야)*  


### **데이터 분석의 목적**  
=> 1. 데이터를 **요약**  2. 대상을 **설명** 3. 새로 얻을 데이터 **예측**  
   
&nbsp;&nbsp;***1) 데이터 요약***  
&nbsp;&nbsp;&nbsp; : Raw data는 수치의 나열일 뿐, 경향을 알기 어려움 -> 요약하여 정리 필요  

&nbsp;&nbsp;***2) 대상 설명***  
&nbsp;&nbsp;&nbsp; : 대상이 가진 **성질과 관계성** 명확히 밝히고 이를 이해 (not 우연히 일어날 가능성)  
&nbsp;&nbsp;&nbsp;&nbsp; * 통계학에 의거한 데이터 분석의 의미 ) 데이터를 **정량적이고 객관적**으로 평가하여 대상이 가진 **성질과 관계성**을 찾는 것  

> 📂 <u>*(case) 신약 효과를 보기 위해 고혈압 환자를 신약집단과 위약 집단으로 나누고 약을 복용한 뒤, 혈압 측정*</u>  
> 
>* **실험군** : 실험에서 효과가 기대되는 처리를 한 집단  
&nbsp; ex) 신약을 투여한 집단
>* **대조군 or 통제군** : 이와 비교,대조하기 위한 집단  
&nbsp; ex) 위약을 투여한 집단  
> > **단일맹검법** : 피험자에게 어느 쪽 약인지 알리지 않고 실험하는 방법  
> > **이중맹검법** : 연구자도 어느 쪽 약인지 모른 채 실험하는 방법
> ---  
> (Why?) 편견 등으로 결과에 영향을 줄 "모든 가능성을 배제"해 확고한 증거를 얻기 위함

<br>

> 📂 <u>*"설명에는 수준이 있다"*</u>  
> * **인과관계** : 2가지 중 하나(원인)만을 변화시키면, 다른 하나(결과)도 바꿀 수 있는 관계  
   ex) 신약에 포함 된 특정 성분이 혈압을 내리는 데 효과가 있음  
   +) 개입 : 원인을 바꾸는 것 
> * **상관관계** : 한쪽이 크면 다른 한 쪽도 큰(또는 한쪽이 크면 다른 한 쪽이 작은)관계  
   +) 인과관계와의 차이) 한쪽을 변화시킨다고 해서 다른 한쪽이 변한다고 단정할 수 없음  
   ex) 연소득과 행복도간의 관계


&nbsp;&nbsp;***3) 미지의 데이터 예측***  
&nbsp;&nbsp;&nbsp; : 이미 얻은 데이터를 기반으로 새롭게 얻을 데이터 예측  
&nbsp;&nbsp;&nbsp;&nbsp; * 예측과 실제에는 차이 생기기 마련 -> 통계분석을 통해 오차를 줄이는 관계성을 발견해야 함  
&nbsp;&nbsp;&nbsp;&nbsp; * 기계학습 : 해석이 어려운 복잡한 관계를 추출하고 예측

## 02. 통계학의 역할

```
✅ 학습 목표 :
* 산포에 대해 설명할 수 있다.
```
### **퍼짐(산포, Dispersion)**  
: 데이터에 포함된 값 하나하나의 차이  
&nbsp; -> 데분에서 통계학의 역할은 **퍼짐**이 있는 데이터에 대해 **설명이나 예측**을 하는 것  
- 데이터 퍼짐은 대상이 가진 성질이나 관계성의 본모습을 감춤
- 통계학은 이러한 데이터 퍼짐을 **불확실성**이라고 평가  

### **확률론을 사용하는 이유** 
: **확률론**을 통해 데이터 퍼짐이나 불확실성을 확률로 나타내어 대처  
<br>

## 03. 통계학의 전체 모습
```
✅ 학습 목표 :
* 기술통계와 추론통계의 차이점을 설명할 수 있다.
* 통계적 추론과 가설 검정에 대해 이해한다.
```
### **기술통계 Descriptive Statistics**  
: 수집한 데이터를 정리하고 요약하는 방법  
- ex) 평균, 분산 등 
- 데이터 그 자체의 특성과 경향
- 확보한 데이터 자체의 성질 이해가 목표  

### **추론통계 Inferential Statistics**  
: 수집한 데이터로부터 데이터의 **발생원**을 추정하는 방법
- **"확률모형과 추정"**  
&nbsp;(Problem) 대상의 성질 자체는 관찰할 수 없으면 다루기 어려움  
&nbsp; -> (Solution) 데이터가 비교적 단순한 확률장치(***확률모형***)에서 생성되었다고 가정  
- 추론통계의 2가지 종류  
&nbsp;&nbsp;***1. 통계적 추론(statisitical inference)***  
&nbsp;&nbsp;&nbsp;: 데이터에서 가정한 확률모형의 성질을 추정하는 방법  
&nbsp;&nbsp;***2. 가설검정(statistical test)***  
&nbsp;&nbsp;&nbsp;: 세운 가설과 얻은 데이터가 얼마나 들어맞는지를 평가해 가설을 채택할 것인가를 판단하는 방법  
<br>

```
✅ 학습 목표 :

```

데이터 분석의 첫단계 ***"분석의 목적과 알고자하는 대상"*** 정하기  
&nbsp; ex) 신약효과확인 case : 데이터 분석 대상은 모든 고혈압 환자 (*not 실험참가인원*)  
<br>

## 05. 모집단

```
✅ 학습 목표 :
* 모집단의 개념을 명확히 설명할 수 있다.
```
### **모집단**  
: 알고자 하는 대상 전체  
&nbsp; ex) 신약효과확인 case : 고혈압 환자 중 신약 복용자 포함 모집단 / 위약 복용자 포함 모집단  
- 실제로 데이터를 얻을 가능성이 없는 요소를 포함한 모집단은 적절X  

### **모집단크기**  
: 모집단에 포함된 요소(element)의 수  
- ***유한모집단*** : 모집단 중 한정된 요소만 포함  
&nbsp;&nbsp; ex) 광고홍보학과 4학년 학생
- ***무한모집단*** : 모집단 중 포함된 요소의 개수가 무한함  
&nbsp;&nbsp; ex) 주사위의 눈  
<br>

## 06. 모집단의 성질을 알다 

```
✅ 학습 목표 :
* 전수조사와 표본조사의 개념적 차이와 각각의 필요성을 설명할 수 있다.
* 표본에 대해 이해한다.
```
### **모집단성질**  
*"모집단의 성질을 알 수 있다면 대상을 설명하거나 이해할 수 있고, 미지의 데이터를 예측할 수도 있게됨"*  

&nbsp; **[ 모집단성질의 예 ]**  
&nbsp;&nbsp; - 신약을 복용한 사람의 최고 혈압 평균은 120mmHg이다.  
&nbsp;&nbsp; - 이 주사위는 6의 눈이 1/4 확률로 나온다.

### **전수조사**  
: 모집단에 포함된 **모든 요소를 조사**해 모집단의 성질을 파악하는 방법  
&nbsp; ex) 인구주택총조사  
- **유한모집단**일 때 선택 가능  
- ***"분석할 데이터 = 모집단"***  
- (Pros) 획득한 데이터의 **기술통계**를 통해 모집단 성질 이해 가능
- (Cons) 시간/비용 면에서 실현불가능성 높음  

### **표본조사**  
: 모집단에서 뽑은 **표본**을 이용해 모집단의 성질을 조사하는 것  
&nbsp; ex) 선거출구조사  
- 유한모집단/무한모집단 모두 선택 가능  
- (Pros) 모집단 전체를 조사하지 않아도 **추론통계**를 통해 모집단의 성질을 알아낼 수 있음
- (Cons) 모집단의 성질 100% 알 수X -> 불확실성 염두  

### **표본크기 Sample Size**  
: 표본에 포함된 요소의 개수(n)  
=> 모집단의 성질을 추정할 때의 확실성이나 가설검정의 결과에도 영향을 미침

> ⚠️ <u>**표본크기 VS 표본 수**</u>  
> *(case) 20명으로 이루어진 표본 A / 30명으로 이루어진 표본 B*  
> - 표본크기 : A는 20, B는 30
> - 표본수(샘플수) : 2

# 2️⃣ 확인 문제

## 문제 1.

> **🧚Q. 한 회사가 직원들의 ‘업무 만족도’를 파악하기 위해 설문조사를 실시했다.
전체 직원 1,000명 중 50명에게만 설문을 진행했고, 그 결과 평균 만족도는 4.5점(5점 만점)이었다.
회사는 이를 근거로 “우리 회사 직원들은 대체로 업무 만족도가 높다”고 결론 내리려 한다.**
>
> **이 결론이 타당한지 판단하기 위해 가장 먼저 확인해야 할 것은 무엇인가?**

~~~
1️⃣ 조사 대상 50명이 전체 직원을 대표하는 표본인가?
2️⃣ 만족도의 평균값만으로 결론을 내려도 되는가?
3️⃣ 설문 문항이 객관적으로 구성되었는가?
4️⃣ 이 결과를 모집단(전체 직원)으로 일반화해도 되는가?
~~~




```
4장에서 알 수 있듯이 데이터 분석의 첫단계는 "분석의 목적과 알고자하는 대상" 정하기이다. 문제에서 제시된 분석의 목적은 직원들의 업무 만족도를 파악하는 것으로, 알고자하는 대상(모집단)은 회사의 직원 1000명 이라고 할 수 있다. 1000명 중 50명을 뽑아 표본조사를 진행한 것으로 표본조사 시에는 추론통계를 통해 모집단의 성질을 알 수는 있지만 100%을 알 수는 없기 때문에 불확실성을 염두해둬야 한다. 따라서 1️⃣ 과 같이 표본의 대표성에 대해서 확인하는 것이 우선이라고 생각한다. 표본크기는 적당한지, 표본이 편향(예를 들어 특정 부서에서만 50명을 뽑음) 등 모집단을 대표하는 표본을 선택했는지가 통계분석의 첫 단계라고 생각한다. 
```
